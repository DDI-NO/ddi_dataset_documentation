# DDI Dataset Documentation

This site documents the dataset and variables available in the Dementia Disease Initiative (DDI) database exports. 

The documentation follows the same logic as the data collection and organization in the DDI project, which is structured around several several categories of data modalities and biological domains, including:

- Clinical data
    - [Demographics](experiments/demographics.md)
    - [Medical history](experiments/medhist.md)
    - [Cognitive assessments](experiments/cognitive_screening.md)
    - [Physical examinations](experiments/physical_exam.md)
    - [Cognitive staging and diagnosis](experiments/diag_diagnosis.md)
- Fluid biomarkers
    - [Blood samples](experiments/blood_biomarkers.md)
    - [Cerebrospinal fluid (CSF) samples](experiments/csf_biomarkers.md)
- Neuroimaging-derived biomarkers (documentation in progress)
    - MRI radiology read
    - PET radiology read
    - Brain volumes
    - Cortical thickness
- Genetic data
    - APOE Genotyping (included in blood biomarker data)
    - Polygenic Risk Scores (Not available in default dataset export)
    - Genome-Wide Association Studies (Not available in default dataset export)

You can navigate through the experiments or the full flat dataset. The experiments are organized by data modality and biological domain, to explore the specific variables collected in each category. Each experiment page provides detailed information about the variables, including their definitions, data types, and any relevant metadata. The full flat dataset view provides a comprehensive catalogue of all variables available in the dataset, allowing for easy reference and exploration of the entire dataset.  

The search functionality allows you to quickly find specific variables or topics across the entire documentation, making it easier to navigate and access the information you need.

## Brief Project Description

The Dementia Disease Initiation project is a Norwegian longitudinal multicenter research initiative focused on understanding the early stages and progression to dementia. The project collects a wide range of data, including clinical assessments, cognitive testing, neuroimaging, biomarkers, genetic information, and lifestyle factors, to identify biomarkers and risk factors associated with the onset of dementia. Data collection follows a well-defined follow-up protocol, with participants undergoing two-year follow-up assessments up to ten years or until the onset of dementia. 

At each research visit, clinical data is collected using a standardized case report form (CRF), neuroimaging data is acquired using a standardized MRI protocol, and blood and CSF biospecimens are drawn for biomarker and genetic analyses. The collected data is recorded in a secured customized XNAT instance, which serves as the central repository and database for all project data.
